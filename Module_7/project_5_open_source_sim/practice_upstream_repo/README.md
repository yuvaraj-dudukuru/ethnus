# 📦 Awesome Snippets (practice "upstream" repo)

> **This is a deliberately tiny project for you to practice contributing to.**
> Treat it as if *you don't own it*. Push it to GitHub under a "teammate's"
> account (or a second account of yours), then go through the full
> fork → upstream → PR flow against it before you try a real project.

A small collection of handy code snippets. Contributions welcome — especially
fixing typos and adding new snippets!

## Snippets
See [snippets.md](snippets.md).

## How to contribute
Read [CONTRIBUTING.md](CONTRIBUTING.md) (the one in the parent folder is a good
template). In short: fork → branch → `git commit -s` → PR.

## Good first issues (pretend these are GitHub issues)
- **#1** Typo in `snippets.md`: "fucntion" should be "function".
- **#2** Add a snippet for "reverse a string in Python".
- **#3** The README is missing a License section.

Pick one, fix it on a branch, and open a PR. 🎯
