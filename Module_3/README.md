# Module 3 — Front-End Web Development (HTML, CSS, Bootstrap & JS)

This module moves from the terminal into the **browser**. It is a collection of
**six progressively harder website projects** that together teach modern front-
end web development — from hand-writing your first HTML/CSS page all the way to a
multi-page, responsive business site.

Each project lives in its **own folder** and has its **own detailed README**.
Start at Project 1 and work your way up — each one introduces something new while
reinforcing what came before.

## 🧱 The three languages of the front end

Everything in this module is built from three technologies that work together:

| Language | Job | Analogy |
|----------|-----|---------|
| **HTML** | Structure & content (headings, text, images, links) | The **skeleton** |
| **CSS** | Appearance (colours, layout, spacing, animation) | The **skin & clothes** |
| **JavaScript** | Behaviour (reacting to clicks, validating forms) | The **muscles** |

Projects 1–3 focus on HTML & CSS. Project 4 adds JavaScript. Projects 5–6 combine
all three into realistic pages.

---

## 🎨 What is Bootstrap (used in Projects 2–6)?

**Bootstrap** is a free **CSS framework** — a huge, ready-made stylesheet of pre-
designed classes. Instead of writing all your own CSS, you add class names like
`btn`, `navbar`, `card`, `row`, and `col-md-6` to your HTML and they instantly
look polished and professional. It's loaded with a single `<link>` from a CDN
(a fast online file host), so there's nothing to install.

Its most important feature is the **12-column responsive grid**, which makes
layouts automatically rearrange to fit phones, tablets, and desktops.

## 📈 Concept progression across the module

```
Project 1  ──►  Raw HTML + CSS. Learn the fundamentals by hand.
Project 2  ──►  Discover Bootstrap. Style a whole page with zero custom CSS.
Project 3  ──►  Bootstrap + your own CSS. Add a modal & embedded map.
Project 4  ──►  Add JavaScript: a self-validating registration form.
Project 5  ──►  Assemble a full e-commerce product page.
Project 6  ──►  Think bigger: a multi-page site with shared layout.
```

By the end you will have touched: semantic HTML, CSS variables, Flexbox, CSS
Grid, transitions & `@keyframes` animations, **responsive design** with media
queries, the Bootstrap grid and components (navbars, cards, badges, tables,
modals, forms), Bootstrap Icons, embedded maps, and basic JavaScript event
handling and form validation.

## ▶️ How to view any project

These are static websites — **no build step or server needed**:

1. Open the project's folder.
2. **Double-click the `.html` file** (e.g. `index.html`) to open it in your
   default web browser.
3. **Resize the browser window** to see the **responsive** design adapt.

> 💡 For a better editing experience with auto-refresh, install the **Live
> Server** extension in VS Code, then right-click the HTML file → *Open with
> Live Server*.

> 📁 Some projects use an `images/` folder. Keep those images next to the HTML
> file or the photos/backgrounds won't show.
